# @mcbe-toolbox-lc/sukuriputils

[![NPM Version](https://img.shields.io/npm/v/%40mcbe-toolbox-lc%2Fsukuriputils)](https://www.npmjs.com/package/@mcbe-toolbox-lc/sukuriputils)
[![GitHub Tag](https://img.shields.io/github/v/tag/mcbe-toolbox-lc/sukuriputils)](https://github.com/mcbe-toolbox-lc/sukuriputils/tags)

A library containing many utility functions for Minecraft Bedrock scripting.

Requires `@minecraft/server@^2.3.0`.

## Usage

Run the following command in your project directory to install sukuriputils:

```bash
npm install @mcbe-toolbox-lc/sukuriputils --save-dev
```
